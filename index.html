What You'll See

1. The maze can run horizontally or vertically. Randomly generated walls will appear depending on the orientation.
2. A green "entry" point will be placed at an edge.
3. A red "exit" point will be placed at an edge.
4. The search will begin from the entry, seeking the exit. The color changes with each step, displaying the distance the search has travelled.
5. The search stops when the exit is found.
6. The optimal (shortest) path from exit to entry is displayed. The distance (in steps) is displayed on the path as it is revealed.
